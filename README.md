## 考试内容

信息安全工程师考试考试科目共有二门：

基础知识，上午考试，考试时间为150分钟，笔试，75道选择题，总分75分，45分及格；
应用技术，下午考试，考试时间为150分钟，笔试，问答题，一般为4道题目，总分75分，45分及格；
